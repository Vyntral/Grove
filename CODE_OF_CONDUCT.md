# Code of conduct

Grove follows the
[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be civil. Disagree with ideas, not with people.
- Default to charitable readings of others' work.
- No harassment, no bigotry, no doxxing — instant ban.
- If you see something off, email **luca.lorenzi@orizon.one** rather
  than confronting publicly. Mediation works better in private.

## Scope

This applies to every project space — issues, PRs, discussions, the
Discord, and any in-person event under the Grove name.

## Enforcement

Maintainers will respond to reports within 7 days. Sanctions, in order:

1. Private warning + a description of the change required.
2. Temporary lock from one or more spaces (1–30 days).
3. Permanent ban.
